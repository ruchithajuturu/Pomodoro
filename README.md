Welcome to the **Pomodoro Timer**! This is a simple and elegant web-based Pomodoro timer designed to help you stay focused and productive. The Pomodoro Technique is a time management method that uses a timer to break work into intervals, traditionally 25 minutes in length, separated by short breaks.

## Live Website

You can access the live website here: [Pomodoro Timer](https://ruchithajuturu.github.io/Pomodoro/)

## Features

- **Pomodoro Timer**: A 25-minute timer to help you focus on your tasks.
- **Pause and Reset**: Pause the timer or reset it to start over.
- **Dynamic Background**: The background updates with a beautiful cherry blossom wallpaper every time the page is refreshed.
- **Motivational Quotes**: Get inspired with a new motivational quote displayed at the bottom of the page every time you refresh.

## Technologies Used

- **HTML**: For structuring the website.
- **CSS**: For styling the website, including gradient text and responsive design.
- **JavaScript**: For implementing the timer functionality, dynamic background, and motivational quotes.

## How to Use

1. Open the [live website](https://ruchithajuturu.github.io/Pomodoro/) in your browser.
2. Click the **Start** button to begin the 25-minute timer.
3. Use the **Pause** button to pause the timer at any time.
4. Click the **Reset** button to reset the timer back to 25:00.
5. Refresh the page to see motivational quote.
